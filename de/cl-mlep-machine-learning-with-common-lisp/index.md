Obwohl Common Lisp als wichtige KI Sprache gilt, kenne ich keine Open Source Machine Learning Bibliothek. [CLML](http://cl-www.msi.co.jp/solutions/knowledge/lisp-world/products/CLML-en.html) scheint umfangreich zu sein, aber Zugriff auf den Quellcode hat man nur gegen Bezahlung. Daher habe ich eine kleine Bibliothek für Machine Learning geschrieben und auf [GitHub](https://github.com/fzalkow/cl-mlep) veröffentlicht. Alle weiteren Details, Anwendungsbeispiele sowie die API findet sich auf der [Projektseite](http://fzalkow.github.io/cl-mlep/).
