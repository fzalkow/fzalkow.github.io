cl-mlep: Machine Learning mit Common Lisp

Auch verfügbar in:

English

Obwohl Common Lisp als wichtige KI Sprache gilt, kenne ich keine Open Source Machine Learning Bibliothek. CLML scheint umfangreich zu sein, aber Zugriff auf den Quellcode hat man nur gegen Bezahlung. Daher habe ich eine kleine Bibliothek für Machine Learning geschrieben und auf GitHub veröffentlicht. Alle weiteren Details, Anwendungsbeispiele sowie die API findet sich auf der Projektseite.
